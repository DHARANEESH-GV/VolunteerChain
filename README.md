# VolunteerChain
VolunteerChain is a decentralized platform that transforms disaster response by intelligently matching verified volunteers to real-time tasks, solving the lethal coordination failures seen in crises like the Uttarakhand floods where doctors sat idle while victims needed aid. Using AI/ML for optimal skill-based assignment and blockchain to immutably verify credentials and contributions, the platform connects volunteers registering with proof of skills (medical license, driver, cook) to control rooms posting specific tasks. The result is a real-time, trustworthy system that deploys the right person to the right place, tracks every contribution, and issues tamper-proof digital certificates—effectively creating a "LinkedIn for disaster volunteers" that builds an immutable reputation and trust for future employers and NGOs.
